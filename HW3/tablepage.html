Create custom multiplication tables!

Using the table below, add two x values and two y values regradless of order as long as the values are valid.

The values that are not valid include the following:

  • Non-numbers
  • Values over 100
  • Values under -100
  • Any value that is 0
  • Decimal numbers

The table has the following rules:

  • Upper Left most has the smallest values.
  • Values going right will increase in the X axis
  • Values going down will increase in the Y axis
